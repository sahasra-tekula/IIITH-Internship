 Object Detection using YOLOv8

This project demonstrates object detection using a pretrained YOLOv8 model from Ultralytics.

 Steps Performed:
- Created Python virtual environment
- Installed ultralytics package
- Ran object detection on a custom image

 Results:
The model successfully detected:
- Chair
- Monitor (TV)
- Multiple potted plants

Confidence scores are shown for each detected object.

Note:
A custom image was used instead of the default sample to demonstrate real-world usage.
